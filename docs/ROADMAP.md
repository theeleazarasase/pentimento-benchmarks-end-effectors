# Roadmap

## Module 1 — Pull-to-slip rig + force dataset
- Rig build + photos
- HX711 calibration + validation
- Logging + peak extraction
- Dataset: 10 objects × 5 trials each (50 trials)

## Module 2 — Closing force control (servo) + grasp repeatability
- Controlled clamp force
- Pad material comparison
- Planning-aware constraints (approach angle, misalignment)

## Module 3 — Multi-finger prototypes (compliance)
- Interchangeable fingertips
- Surface robustness studies

## Module 4 — Simple perception-assisted grasp selection (PC)
- Object ID → grasp config selection
- Logged failure modes
